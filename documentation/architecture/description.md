Описание регистрации:

Команда /start 

После этой команды сразу создается запись в базе с информацией о пользователе. Заполнены поля id, telegramId и name

Когда пользователь впервые заходит в бота ему предоставляется выбор роли: Игрок или Организатор.

Если Пользователь ранее не регистрировался в системе, то он видит сообщение: "Привет! Я бот для организации волейбольных игр. Выбери свою роль:"

Отображается клавиатура с выбором роли Игрока или Организатора

Если пользователь выбирает создать профиль Игрока то ему предлагается выбрать свой уровень (который пока ни на что не влияет). Пользователь видит сообщение: "Ты выбрал роль игрока. Теперь оцени свой уровень:"

После выбора уровня значение сохраняется в базу
